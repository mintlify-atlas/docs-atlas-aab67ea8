## Overview

The `JobListByPlatform` component displays all scraping campaigns for a specific platform with:

1. Campaign list with statistics (available usernames, assignments)
2. Airtable base links
3. Refresh functionality
4. Navigation to campaign dashboards
5. Create new campaign button

Located at: `components/job-list-by-platform.tsx`

## Props

```tsx
interface JobListByPlatformProps {
  /** Platform to display campaigns for */
  platform: Platform
}

type Platform = 'instagram' | 'threads' | 'tiktok' | 'x'
```

## Features

### 1. Fetch Jobs by Platform

```tsx
import { getJobsByPlatform, getJobStatisticsByBaseId } from '@/lib/scraping-jobs'

async function fetchJobs() {
  const platformKey = platform.toLowerCase() as 'instagram' | 'threads' | 'tiktok' | 'x'
  const jobsData = await getJobsByPlatform(platformKey)
  
  // Fetch stats for each job
  const jobsWithStats = await Promise.all(
    jobsData.map(async (job) => {
      const baseId = job.airtable_base_id || job.base_id
      const stats = baseId 
        ? await getJobStatisticsByBaseId(baseId)
        : null
      
      return {
        ...job,
        stats: stats || {
          totalUsernames: 0,
          usedUsernames: 0,
          availableUsernames: 0,
          totalAssignments: 0,
          totalScrapeResults: 0
        }
      }
    })
  )
  
  setJobs(jobsWithStats)
}
```
