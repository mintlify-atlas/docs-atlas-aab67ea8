# Welcome to Click Creators Scraper

Click Creators Scraper is a comprehensive Instagram marketing automation platform that streamlines follower scraping, profile filtering, campaign management, and VA (Virtual Assistant) distribution through Airtable.

## What is Click Creators Scraper?

Click Creators Scraper automates the entire workflow of sourcing Instagram profiles for marketing campaigns:

1. **Scrape followers** from Instagram influencer accounts
2. **Filter profiles** by gender using name-based detection
3. **Manage campaigns** with daily batches of fresh profiles
4. **Distribute to VAs** across 80 Airtable tables (180 profiles each)
5. **Track progress** with real-time statistics and status monitoring

### Key Metrics

- **Daily Target**: 14,400 unique male profiles
- **VA Distribution**: 80 virtual assistants
- **Profiles per VA**: 180 per day
- **Campaign Lifecycle**: 7 days
- **Total Capacity**: 1.008M profiles/week

## Tech Stack

**Frontend**
- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS + ShadCN UI
- Supabase Client

**Backend**
- Flask API (Python)
- Apify (Instagram scraping)
- Supabase PostgreSQL
- Airtable API

**Database**
- PostgreSQL with Row Level Security
- Real-time subscriptions
- Multi-tenant isolation
